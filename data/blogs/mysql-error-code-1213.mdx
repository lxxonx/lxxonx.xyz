### 내용

update for select 에서 자체적인 lock을 걸고 그로인한 deadlock 으로 발생하는 에러

### 원인

주로 unique key/primary key 가 설정되지 않은 테이블에서 여러 개의 column을 참조하려고할때 오류가 발생

### 해결

참조하고자하는 여러개의 테이블을 묶어서 unique key로 생성한다
