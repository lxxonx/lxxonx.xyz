## 설명

### 디자인

전체적인 디자인은 pixel 디자인의 아이콘과 Galmury 폰트를 사용하여 레트로 느낌을 주었으나\
게시물의 내용은 나눔고딕 폰트를 사용하여 가독성을 높혔다.

### MDX

1. libraries

   - `rehypeSlug`: add id to headings
   - `rehypeAutolinkHeadings`: add link to headings
   - `rehypeHighlight`: syntax highlighting for code blocks\
     github-dark theme is used
   - `rehypeToc`: add table of contents
   - `tailwindcss typography`: mdx styles
   - `next-mdx-remote`: get mdx files from remote directories
   - `reading-time`: get reading time of mdx files

2. load

   - 태그와 관련된 내용은 폴더 안의 모든 게시물을 `gray-matter`를 통해 읽어온 뒤, object로 저장하여 사용한다.
   - 전반적인 스타일은 tailwindCSS의 typography로 하나, code 블럭의 스타일링은 rehypeHighlight와 hightlight.js의 github dark theme으로 하였다.
   - 게시물은 blog/project/snippet으로 분류했으며, 각 폴더별로 따로 읽어온다.
