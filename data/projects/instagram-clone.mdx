## Built With

- [Node.js](https://nodejs.org/ko/) for backend
- [PostgreSQL](https://www.postgresql.org/) for database
- [Prisma ORM](https://www.prisma.io/) as orm
- [Create React App](https://create-react-app.dev/)
- [Apollo Graphql](https://www.apollographql.com/) 상태관리/Graphql
- [Styled Components](https://styled-components.com/) for css
- [Heroku](https://https://www.heroku.com/) for be deployment
- [Vercel](https://vercel.com/) for fe deployment
- [Cloudinary](https://cloudinary.com/) for image-upload

## 설명

- 추후에 어플 또한 만들 계획이었기 때문에 over-fetching/under-fetching 문제를 해결할 수 있는 `graphql`을 사용하여 api를 만들었다.
- `Redis pubsub` 서버를 통해서 메시지 서버를 구현하였으며 타입정의 및 소켓 연결을 위하여 `Graphql Subscription`을 함께 사용하였다.
- 서버에서 상태를 저장할 필요가 없었기 때문에 Session 대신 `JWT`를 통해서 사용자 인증시스템을 구현하였다.
- 메일 인증을 사용하기 위해서 `nodemailer`를 사용하였다.
- 이미지 업로드 저장소로 `cloudinary`를 사용하였다.
  > AWS S3로 구현할 경우 이미지 저장 및 Crop에 대한 함수를 따로 Lambda함수로 구현해야하는 문제가 있어서 시간적인 제약으로 인해서\
  > S3 대신 cloudinary에서 제공하는 sdk를 통해서 이미지를 crop하고 저장하였다.
